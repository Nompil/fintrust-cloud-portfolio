\# Week 3 - Python Standard Library



\## Objective



The purpose of this week's work was to learn how Python interacts with the operating system and file system using built-in modules.



\## What I Built



\### fintrust\_utils.py

Reusable utility functions for:

\- currency formatting

\- account validation

\- transaction summaries



\### setup\_data\_dirs.py

A script that automatically creates the directory structure used by FinTrust data processing projects.



\## Challenges



Understanding how pathlib paths differ from traditional string-based paths was initially confusing, but using Path objects made the code easier to read and more portable.



\## AWS Connection



I learned that environment variables are commonly used in AWS Lambda and other cloud services to store configuration values securely.



\## Reflection



The most valuable learning outcome was understanding how reusable scripts can be configured through command-line arguments and environment variables rather than hard-coded values.



\## Skills Practiced
